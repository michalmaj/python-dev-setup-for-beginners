# Python Dev Setup for Beginners

A beginner-friendly guide to setting up a modern Python development environment from zero.

This repository is for people who are new to programming tools and want to learn how to:

- open and use a terminal,
- understand basic command-line commands,
- install package managers such as Chocolatey, Homebrew, or Linux package managers,
- install Git,
- create and configure a GitHub account,
- connect Git with GitHub,
- install Visual Studio Code,
- create a Python project with uv,
- open and run the project in VS Code.
