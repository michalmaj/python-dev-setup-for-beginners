# Python Dev Setup for Beginners

Przewodnik dla początkujących pokazujący, jak od zera przygotować nowoczesne środowisko do pracy z Pythonem.

Repozytorium jest przeznaczone dla osób, które dopiero zaczynają i chcą nauczyć się:

- jak uruchomić i używać terminala,
- czym są podstawowe komendy terminalowe,
- jak zainstalować menedżery pakietów, takie jak Chocolatey, Homebrew lub menedżery pakietów w Linuxie,
- jak zainstalować Git,
- jak założyć i skonfigurować konto GitHub,
- jak połączyć Git z GitHubem,
- jak zainstalować Visual Studio Code,
- jak utworzyć projekt Pythonowy z użyciem uv,
- jak otworzyć i uruchomić projekt w VS Code.
