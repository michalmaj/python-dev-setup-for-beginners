# Start Here

Welcome to **Python Dev Setup for Beginners**.

This guide is for people who want to start learning programming, Python, Git, GitHub, and Visual Studio Code, but do not yet feel comfortable with developer tools.

The goal is not only to install tools, but also to understand what is happening at each step.

## Who is this guide for?

This guide is for complete beginners who may not yet know:

- what a terminal is,
- how to open PowerShell, Terminal, or a Linux shell,
- how to move between folders using commands,
- what Git is,
- what GitHub is,
- how Git and GitHub are different,
- how to install Visual Studio Code,
- how to create a Python project,
- how to run a project from the terminal,
- how to publish code on GitHub.

No previous programming experience is required.

## What will you learn?

By the end of this guide, you should be able to:

- open a terminal on your operating system,
- use basic terminal commands,
- install basic developer tools,
- install and configure Git,
- create a GitHub account,
- connect your local Git repository with GitHub,
- install Visual Studio Code,
- create a Python project with `uv`,
- open the project in VS Code,
- run Python code from the VS Code terminal,
- make your first commit,
- push your project to GitHub.

## Recommended learning order

This repository is organized into several learning paths.

Start with the path for your operating system:

1. Windows
2. Linux
3. macOS

After that, continue with the shared sections:

1. Git and GitHub
2. Visual Studio Code
3. uv and Python projects
4. First commit and push
5. Troubleshooting

## Learning paths

### Windows path

Recommended for people using Windows 10 or Windows 11.

You will learn how to:

- open PowerShell,
- use basic PowerShell commands,
- install Chocolatey,
- install Git,
- install Visual Studio Code,
- install `uv`,
- create your first Python project.

### Linux path

Recommended for people using Ubuntu, Debian, or similar Linux distributions.

You will learn how to:

- open the terminal,
- use basic shell commands,
- install tools with the system package manager,
- install Git,
- install Visual Studio Code,
- install `uv`,
- create your first Python project.

### macOS path

Recommended for people using macOS.

You will learn how to:

- open Terminal,
- use basic shell commands,
- optionally install Homebrew,
- install Git,
- install Visual Studio Code,
- install `uv`,
- create your first Python project.

## Important idea

This guide moves slowly on purpose.

Many beginner tutorials start with commands like:

```bash
git clone ...
cd project
code .
```

But they often do not explain what these commands mean.

Here, each step should answer three questions:

1. What are we doing?
2. Why are we doing it?
3. How can we check that it worked?
